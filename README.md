## üïπÔ∏è Usage
- **Space**: Start sort <br>
- **Backspace**: Stop sort <br>
- **h**: Display help <br>
- **F1**: Change number of elements <br>
- **F2**: Change time between comparisons <br>
- **Arrow Up / Arrow down**: Change sort type <br>
